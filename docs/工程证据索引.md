## API 路由

| 路由 | 说明 |
|------|------|
| `POST /jobs` | 创建任务 |
| `GET /jobs` | 列出任务 |
| `GET /jobs/{id}` | 查询任务 |
| `POST /jobs/{id}/schedule` | 手动调度 |
| `POST /jobs/{id}/run` | 调度并执行 |
| `POST /jobs/{id}/retry` | 重试 |
| `POST /jobs/{id}/cancel` | 取消 |
| `GET /jobs/{id}/executions` | 执行记录 |
| `GET /jobs/{id}/trace` | trace 时间线 |
| `POST /dispatch/once?limit=N` | 批量调度 |
| `GET /metrics` | Prometheus 指标 |
| `GET /healthz` | 健康检查 |

**合计：12 条路由**

## 执行器

| 执行器 | 实际行为 |
|--------|----------|
| `shell` | `os/exec.Command` |
| `k8s-dry-run` | 构建 K8s Job YAML，不提交 |
| `k8s-apply` | 需 `ALLOW_K8S_APPLY=true` |
| `http` | 发起 HTTP 调用 |
| `benchmark` | 调用 `internal/benchmark.Run()` |

**合计：5 类执行器**
