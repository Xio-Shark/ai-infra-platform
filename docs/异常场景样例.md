# AI Infra Platform｜异常场景样例

## 1. 非法 job_type

```bash
curl -s -X POST http://localhost:8080/jobs \
  -H 'Content-Type: application/json' \
  -d '{"name":"bad","job_type":"unknown","executor":"shell","command":["echo","hi"]}'
```

```json
{"error": "invalid job_type: unknown, must be training|inference|benchmark"}
```

## 2. 空 command

```bash
curl -s -X POST http://localhost:8080/jobs \
  -H 'Content-Type: application/json' \
  -d '{"name":"empty-cmd","job_type":"training","executor":"shell","command":[]}'
```

```json
{"error": "command must not be empty"}
```

## 3. 不存在的 Job ID

```bash
curl -s http://localhost:8080/jobs/nonexistent-id
```

```json
{"error": "job not found: nonexistent-id"}
```

## 4. 重复取消已取消的 Job

```bash
# 第一次取消：成功
curl -s -X POST http://localhost:8080/jobs/{id}/cancel
# {"status": "cancelled"}

# 第二次取消：状态转换非法
curl -s -X POST http://localhost:8080/jobs/{id}/cancel
# {"error": "invalid state transition: cancelled → cancelled"}
```

## 5. 对已完成 Job 重试

```bash
curl -s -X POST http://localhost:8080/jobs/{id}/retry
```

```json
{"error": "invalid state transition: succeeded → retrying"}
```

## 6. Shell 执行器命令失败

```bash
curl -s -X POST http://localhost:8080/jobs \
  -H 'Content-Type: application/json' \
  -d '{"name":"fail","job_type":"training","executor":"shell","command":["false"]}'
# 创建成功

curl -s -X POST http://localhost:8080/jobs/{id}/run
```

```json
{
  "execution_id": "exec-xxxx",
  "status": "failed",
  "error": "exit status 1",
  "logs": ""
}
```

## 7. K8s apply 未启用

```bash
# 当 ALLOW_K8S_APPLY 未设置时
curl -s -X POST http://localhost:8080/jobs \
  -H 'Content-Type: application/json' \
  -d '{"name":"k8s","job_type":"training","executor":"k8s-apply","command":["train"]}'

curl -s -X POST http://localhost:8080/jobs/{id}/run
```

```json
{
  "status": "failed",
  "error": "k8s-apply executor is disabled (set ALLOW_K8S_APPLY=true to enable)"
}
```

## 8. Benchmark executor 无推理服务

```bash
# 当目标地址不可达时
curl -s -X POST http://localhost:8080/jobs/{id}/run
```

```json
{
  "status": "failed",
  "error": "benchmark failed: Post \"http://unreachable:11434/v1/chat/completions\": dial tcp: lookup unreachable: no such host"
}
```
